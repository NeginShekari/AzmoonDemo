# 🤖 حل مسئله: مسیرهای منحصر به فرد در ماتریس با موانع

## 📋 شرح مسئله

این پروژه حل کاملی برای مسئله **Unique Paths with Obstacles** ارائه می‌دهد. در این مسئله:

- یک ربات در گوشه بالا چپ ماتریس قرار دارد
- ربات می‌خواهد به گوشه پایین راست برسد
- ربات فقط می‌تواند به **راست** یا **پایین** حرکت کند
- برخی خانه‌ها **مسدود** هستند (با عدد 1 نشان داده می‌شوند)
- خانه‌های **آزاد** با عدد 0 نشان داده می‌شوند

**هدف:** محاسبه تعداد مسیرهای منحصر به فرد ممکن

## 💡 الگوریتم

### ایده کلیدی:
برای رسیدن به هر خانه `(i,j)` می‌توانیم از دو مسیر بیاییم:
- از خانه بالایی: `(i-1, j)`
- از خانه چپی: `(i, j-1)`

### فرمول:
```
dp[i][j] = dp[i-1][j] + dp[i][j-1]  (اگر خانه آزاد باشد)
dp[i][j] = 0                        (اگر خانه مسدود باشد)
```

### شرایط مرزی:
- `dp[0][0] = 1` (نقطه شروع)
- اگر نقطه شروع یا پایان مسدود باشد → نتیجه = 0

## 📊 مثال عملی

```python
# ماتریس نمونه
grid = [
    [0, 0, 0],
    [0, 1, 0],  # خانه وسط مسدود
    [0, 0, 0]
]

# نتیجه: 2 مسیر منحصر به فرد
# مسیر 1: راست → راست → پایین → پایین
# مسیر 2: پایین → پایین → راست → راست
```

## 📈 پیچیدگی

- **زمانی:** O(m × n)
- **مکانی:** 
  - روش عادی: O(m × n)
  - روش بهینه: O(n)
